# Name

Orgy a tiny organizer. [Demo](http://bugov.net:3456).

# Requires

* sqlite3
* cpan modules: `cpan Mojolicious DBI Time::Local Digest::MD5 DBD::SQLite HTML::CalendarMonthSimple`

# Install

`./orgy install NAME E-MAIL PASS`

# Run

`./orgy daemon` or see more in `./orgy`.
