# Name

Orgy a tiny organizer.

# Requires

* sqlite3
* cpan modules: `cpan Mojolicious DBI Time::Local Digest::MD5 DBD::SQLite`

# Install

`./orgy install NAME E-MAIL PASS`

# Run

`./orgy daemon` or see more in `./orgy`.
